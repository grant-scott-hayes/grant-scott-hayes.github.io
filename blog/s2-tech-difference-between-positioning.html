...the differences between relative, absolute, and fixed positioning (and the two others).

introduction to the postion types

Every part (element) on a webpage has an invisible container (or box) that you can position on the page using the position property. There are five main position types: static, relative, absolute, fixed, and sticky. Each type is a different method for the web programmer to place items on a page for the viewers.

Documents in HTML display in the order of the document flow. The document flow in HTML displays items in order, one after the other. Some items, block items, display on a new line whereas other items, inline items, will display on the same line as other items. That display property can be changed but by default items such as div, paragraph, and lists, are block items (on a new line) whereas, images, links, and inputs are inline items (don't get a new line).

Positioning is controlled by setting the values of the keywords: top, bottom, left and right. Each items has these keywords but they mean different things depending on which type of position you have declared the item to use. For example top measures something different using absolute compared to relative positioning.

position: static

This is the default value for items on a webpage and not generally used except to force the default position on an item.

The behaviour of static items (all items, by default) in the document flow is to display the item in the order they appear in the document flow.

Example

position: relative

A position set to relative is used to display an item in a position that is offset to it's normal default (or static) position. That means relative to ITSELF and not other items; a common misconception.

Within the document flow, the relative position simply displays the item in a different spot but other items won't shuffle around to fill it's hole in the document. This means the document still recognises the space where the original item was on the page (like an invisible block that the next item still respects in terms of size, shape and location).

If a parent item on a web page has a child item positioned within it, then the relative position has to set in the parent item for the child positioning to work.

Example

position: absolute

The absolute postion allows an item to be placed anywhere you want on a web page. The absolute position refers to the placement of the item within a parent element.
